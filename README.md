# Stack Value
A plugin to visually show the value of stacks, to make bank cleanup, and finding value easier.

Items in your inventory, bank, and equipment are colored by what the whole stack is worth, using
five configurable value tiers. Each tier's color and threshold can be set independently, and a tier
can be hidden entirely by setting its color to fully transparent.

## Options
- **Fill** — in addition to outlining an item, shade the item itself in its tier color. Off by default.
- **Fill Opacity** — how strong that shading is, from 0 (none) up to 255 (the item is filled in
  completely, leaving only its shape).
